# Contributing

1. Fork this repo on GitHub.
2. Clone your fork: `git clone https://github.com/<your-username>/agent-distillation-compiler.git`
3. Add upstream: `git remote add upstream https://github.com/<yeshita-username>/agent-distillation-compiler.git`
4. Sync main daily: `git checkout main && git fetch upstream && git merge upstream/main && git push origin main`
5. Branch per task: `git checkout -b <name>/<feature> main`
6. Commit using conventional prefixes: feat / fix / test / docs / chore / refactor
7. Push to your fork, open a PR into `upstream:main`.
